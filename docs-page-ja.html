はじめに

Bundlezaへようこそ！このガイドは、バンドル割引オファー、アップセル製品、マルチアップセル製品、ギフト製品、バナー、カスタムセクションを作成できる強力なShopifyアプリをすばやく使い始めるのに役立ちます。

前提条件

Bundlezaをインストールする前に、以下を確認してください：

  • アクティブなShopifyストア（任意のプラン）
  • Shopifyダッシュボードへの管理者アクセス
  • ストアに在庫とバリアントのある製品が少なくとも1つ

インストール

ShopifyストアにBundlezaをインストールする手順：

ステップ1: Shopify管理者ダッシュボードにログイン

admin.shopify.comでShopify管理者ダッシュボードにログインします。

ステップ2: Bundlezaの検索とインストール

Shopify App StoreでBundlezaを検索：

  1. App Store検索バーに「Bundleza」と入力
  2. 検索結果でBundlezaアプリを見つける
  3. Bundlezaアプリカードをクリックして詳細を表示
  4. アプリの説明、機能、価格を確認
  5. アプリを追加をクリック

ステップ3: 権限の付与

必要な権限を確認して承認：

  1. 要求された権限を注意深く確認
  2. 権限を付与するためにアプリをインストールをクリック
  3. インストールが完了するのを待つ

バンドル

Shopifyマーチャントとして、Bundlezaのバンドル機能は、平均注文値を大幅に向上させる魅力的な製品バンドルを作成するのに役立ちます。補完製品を戦略的に組み合わせ、魅力的な割引を提供することで、顧客がより多くのアイテムを購入するよう促し、収益を増加させながらショッピング体験を向上させます。

バンドル管理

バンドル管理ページは、作成したすべてのバンドルを監督するための中央ハブです。ここでは、バンドルの編集、削除、複製、ステータスの更新を行い、ストアのオファリングを整理し、最新の状態に保つことができます。

バンドル管理へのアクセス

Bundlezaダッシュボードから、バンドルセクションに移動してバンドル管理ページにアクセスします。

バンドルリストとアクション

バンドルリストには、作成したすべてのバンドルが主要な情報と利用可能なアクションとともに表示されます。各バンドルに対して以下のアクションを実行できます：

  • 編集：バンドルの詳細、製品、設定を変更。
  • 削除：ストアからバンドルを完全に削除。
  • 複製：類似オファーのクイックセットアップのためのバンドルのコピーを作成。
  • ステータス更新：ドラフト（顧客から非表示）とアクティブ（ストアで表示）の間で切り替え。

重要

任意のバンドルを編集するには、まずステータスをドラフトに更新する必要があります。

バンドルの作成と更新

ストアの新しいバンドルオファーを作成するか、既存のバンドルを更新します。このページでは、基本的な詳細から高度なスタイリングと製品選択まで、バンドルのすべての側面を設定できます。

新しいバンドルの作成方法

  1. バンドル名を入力：顧客を引きつける説明的で魅力的な名前をバンドルに提供。
  2. 選択タイプを選択：すべての製品、特定の製品、または特定のコレクションからバンドルに製品を含める方法を選択。
  3. 設定を設定：バンドル設定、バリアントオファー、割引オプションを作成して魅力的な価格層を構築。
  4. スタイルをカスタマイズ：ストアのブランディングに合わせて色、フォント、レイアウトを調整。
  5. 製品を選択：製品選択モーダルを使用してバンドルのメイン製品とバリアントを選択。
  6. バリアントオファーを設定：異なるバンドルバリアントの動的オファーを設定。

7. 各オファーのカスタム画像と無料ギフトを選択

バンドルに無料ギフトを選択するには、「無料ギフト」から「ギフトを選択」ボタンを使用します。これにより、製品とそのバリアントを選択できるモーダルが開きます。

ヒント

顧客からのフィードバックに基づいて最適化するために、複雑な設定を追加する前にシンプルなバンドルでパフォーマンスをテストします。

バンドル名

顧客を引きつける説明的で魅力的な名前をバンドルに提供。（必須）

選択タイプ

ストアの任意の製品、特定の製品、または特定のコレクションの製品をバンドルに含める方法を選択。

  • すべての製品：ストアの任意の製品をバンドルに含める。
  • 特定の製品：個別の製品を手動で選択。
  • 特定のコレクション：選択したコレクションのすべての製品を含める。

設定

価格、可用性、表示オプションを含むバンドルのコア設定を設定。

バンドル設定

  • 今日のみ：バンドルの特別な「今日のみ」プロモーションを有効化。
  • バリアント比較価格：異なるバリアントの比較価格を表示。
  • チェックアウトに直接：バンドルから直接チェックアウトを許可。
  • バリアントピッカーを非表示：バンドルページのバリアント選択オプションを非表示。

バンドルバリアントオファー

バンドルバリアントの動的オファーを設定。数量、割引タイプ（例: 固定金額、パーセンテージ、固定価格）、割引値を設定。オプションはデフォルトで選択可能で、さまざまな価格層を作成するために新しいオファーを複数追加できます。

  • 数量：オファーのアイテム数を指定（完全にカスタマイズ可能）。
  • 割引タイプ：固定金額、パーセンテージ、または固定価格から選択。
  • 割引値：割引金額またはパーセンテージを設定（動的で調整可能）。
  • デフォルト選択：顧客のプリ選択オファーとしてマーク。プリ選択の場合、「カートに追加」または「今すぐ購入」ボタンをクリックするとオファーが自動的にカートに追加されます。選択されていない場合、顧客はボックスをチェックして含める必要があります。

ヒント

すべての数量、割引タイプ、値は完全にカスタマイズ可能です。顧客に最適なものを探すためにさまざまなオファーを実験します。

スタイル

ストアのテーマに合わせてバンドルの視覚的外観をカスタマイズ。色、フォント、レイアウト、その他のスタイリング要素を調整。

タイトルスタイル

  • テキストサイズ：バンドルタイトルのフォントサイズを設定（例: 18px）。
  • フォントウェイト：フォントウェイトを選択（太字または通常）。
  • 色：タイトルテキストの色を選択。

サブタイトルスタイル

  • テキストサイズ：サブタイトルのフォントサイズを設定（例: 16px）。
  • フォントウェイト：フォントウェイトを選択（太字または通常）。
  • 色：サブタイトルテキストの色を選択。

価格色

  • 色：通常価格の色を設定。
  • 比較価格色：取り消し線付き比較価格の色を設定。

バッジ色

  • 背景：バッジの背景色。
  • テキスト：バッジのテキスト色。

ラベル色

  • 背景：ラベルの背景色。
  • テキスト：ラベルのテキスト色。

ギフト色

  • 背景：ギフト要素の背景色。
  • テキスト：ギフト要素のテキスト色。

ボックス色

  • ボーダー色：ボックスのボーダー色。
  • ボックス色：ボックスの背景色。
  • 選択ボックス色：選択ボックスの背景色。
  • ボックスボーダー幅：ボックスのボーダー幅（例: 1px）。
  • ボックスボーダーラジアス：ボックスのコーナーラジアス（例: 0px）。

画像設定

  • 画像サイズ：製品画像のサイズを設定（例: 2）。
  • 半径：画像のコーナーラジアス（例: 0px）。

ギフト製品 - 画像設定

  • 画像サイズ：ギフト製品画像のサイズを設定（例: 2）。
  • 半径：ギフト画像のコーナーラジアス（例: 0px）。

追加オプション

  • 今日のみ：特別な「今日のみ」メッセージングを有効化。
  • 最も人気：バンドルを最も人気のあるものとしてマーク。
  • シングル：シングル購入の標準価格。
  • デュオ：デュオ購入の標準価格。
  • 🌐：国際化設定。

    • 国際化設定により、バンドルテキストを1つのカスタム言語に翻訳するか、デフォルトの英語のままにします。

製品選択: このバンドルを表示する場所

製品選択モーダルを使用してバンドルの製品を選択。メイン製品または特定のバリアントを選択できます。

  • メイン製品：バンドルの主要製品を選択。
  • バリアント：より正確な制御のための特定のバリアント（例: サイズ、色）を選択。

バリアントオファー

バンドル内の製品に異なるオプションを選択できるようにバリアント固有のオファーを設定。

  • バリアント選択：顧客がバンドルで製品バリアントを選択できるように有効化。
  • 価格調整：選択されたバリアントに基づいて異なる割引を適用。
  • 可用性ルール：バンドルで利用可能なバリアントのルールを設定。

注意

すべてのスタイル設定は動的で、ブランドに合わせて調整できます。デフォルトは開始点として提供されますが、すべてがマーチャント向けにカスタマイズ可能です。

テーマへのバンドルの追加

作成したバンドルをストアの製品ページに表示するには、ShopifyテーマにBundlezaバンドルセクションを追加する必要があります。これにより、顧客が製品ページで直接バンドルオファーとやり取りできるようになります。

バンドルセクションの追加手順

  1. オンラインストアに移動：Shopify管理者からオンラインストア > テーマに移動。
  2. テーマをカスタマイズ：ライブテーマの横にあるカスタマイズボタンをクリック。
  3. 製品ページを選択：テーマエディタでページセレクタを使用して製品ページテンプレートを選択（例: デフォルト製品）。
  4. セクションを追加：左側のサイドバーでセクションを追加をクリックし、AppsセクションからBundlezaバンドルを検索して選択。
  5. セクションを設定：追加したら、セクション設定のドロップダウンから表示する特定のバンドルを選択。
  6. 変更を保存：テーマに変更を適用するために保存をクリック。

ヒント

Bundlezaバンドルセクションを複数の製品テンプレートやコレクションページにも追加して、より広い可視性を確保できます。ストアのデザインに適合することを確認するために配置をテストします。

シングルアップセルの作成

新しいシングルアップセルの作成方法

アップセル名

顧客を引きつける説明的で魅力的な名前をアップセルに提供。（必須）

可視性

アップセルを表示する場所を選択。オプションにはすべての製品、特定の製品、または特定のコレクションが含まれます。

  • すべての製品：ストアの任意の製品にアップセルを表示。
  • 特定の製品：アップセル表示のために特定の製品を選択。
  • 特定のコレクション：選択したコレクションの製品にアップセルを表示。

製品選択: このアップセルを表示する場所

アップセルが表示される製品またはコレクションを選択。

設定

価格と表示オプションを含むアップセルのコア設定を設定。

  • 画像を表示：製品画像表示を有効または無効。
  • チェックアウトにスキップ：アップセルから直接チェックアウトを許可。
  • 製品比較価格をストライクスルー：取り消し線付き比較価格を表示。

  3. 選択を確認してそのテーマのセクション追加を有効化。

セクションを追加

テーマを選択したら、追加したいセクションの「追加」ボタンをクリックします。または、ガイダンスのためにビデオチュートリアルを見ることもできます。

セクションを追加する方法

  1. テーマで利用可能な事前構築済みセクションのリストを確認。
  2. 含めたいセクション（例: レビューボックス、ベネフィットボックス）の横にある「追加」ボタンをクリック。
  3. コンテンツとスタイリングなどのセクション設定を設定。
  4. 変更を保存し、製品ページでセクションをプレビュー。

マーチャントとして、製品ページにカスタムセクションを追加することで、主要なベネフィット、顧客レビュー、バンドルオファーを強調し、購入決定に直接影響を与え、コンバージョンを改善します。

新しいバナーの作成

重要

バナーを作成する前に、セクションに移動してテーマに追加する必要があります

プレビュー

バナー作成ページには、デスクトップ、モバイル、タブレットデバイスでバナーがどのように見えるかを確認できるライブプレビュー機能が含まれています。これにより、保存前にすべての画面サイズでバナーが最適化されていることを確認できます。

テンプレート、テーマ、スケジューリング、コンテンツオプションでバナーをデザインおよびカスタマイズして、コンバージョンを向上させる魅力的なプロモーションを作成します。

テンプレート

バナーの作成をすばやく開始するために、事前デザインされたテンプレートを選択。

テーマ

ブラックフライデー、クリスマス、ニューイヤー2025、コンパクトモード、バレンタインなどから季節プロモーションに合わせてテーマを選択。各テーマにはどのように見えるかのプレビューが含まれます。

  • ブラックフライデー：ブラックフライデーテーマのプレビュー
  • クリスマス：クリスマステーマのプレビュー
  • ニューイヤー2025：ニューイヤー2025テーマのプレビュー
  • コンパクトモード：コンパクトモードテーマのプレビュー
  • バレンタイン：バレンタインテーマのプレビュー

スケジュール

開始日時と終了日時を設定してバナーがアクティブになるタイミングを決定。例えば、2025-09-23 09:00 AMに開始し、2025-09-23 09:00 AMに終了。開始時間が選択されるとバナーが開始することを覚えておいてください。

言語

グローバルオーディエンスにリーチするために異なる言語でバナーコンテンツをカスタマイズ。

バナーコンテンツ

バナーのテキストをここでデザイン：

  • メインテキスト：ブラックフライデー - バナーの左側に目立つように表示。
  • コールトゥアクション：BUNDLEZA50 - 顧客が使用できる割引コード、例: SALE50。
  • カウントダウンラベル：カウントダウン - カウントダウンタイマーの上にあるテキスト。
  • 割引告知：コードで最大50%オフ - 例: 'すべてで{{50%}}オフ'。
  • 閉じるボタン：閉じた後の動作を定義。
  • 表示間隔：閉じた後にバナーが再表示されるタイミングを定義（常に表示）。

デザイン

ブランドデザインに合わせてバナーの外観をカスタマイズ：

  • バナー：バナーテキスト（左）、バナーアクセント色、バナー背景。
  • カウントダウン：テキスト、カウンターテキスト、カウンターバックグラウンド。
  • 割引：割引コードのコピーボタンのテキスト（コードテキスト）、告知コードテキスト、コード背景、コード半径（例: 4）。

スタイル

視覚要素を正確に制御するためにバナーのスタイルをさらにカスタマイズし、色を含みます。 これらの設定により、ブランドパレットに合わせたメインテキスト、カウントダウンタイマー、割引コードテキストの色を調整できます。例えば、見出しにプライマリブランド色を使用し、二次テキストにコントラスト色を使用します。

  • テキスト色：メインテキスト、カウントダウンタイマー、割引コードテキストの色をブランドパレットに合わせて調整。例えば、見出しにプライマリブランド色を使用し、二次テキストにコントラスト色を使用します。
  • 背景色：バナー、カウントダウンセクション、割引コードエリアのカスタム背景を設定。デザインテーマに合わせてソリッドカラー、グラデーション、または透明背景を選択。
  • ボーダーラジアス：割引コードボックスやボタンなどの要素のコーナーラジアスを設定。高い半径はモダンな丸みを帯びたコーナーを作成し、0pxはクラシックなシャープエッジを与えます。

成功

バナーは2025-09-23 09:00から2025-09-23 09:00までアクティブになります

テーマへのバナーの追加

設定したバナーをストアのページに表示するには、ShopifyテーマにBundlezaバナーセクションを追加する必要があります。これにより、製品ページ、ホームページ、または他のテンプレートでバナーが表示されます。

バナーセクションの追加手順

  1. オンラインストアに移動：Shopify管理者からオンラインストア > テーマに移動。
  2. テーマをカスタマイズ：ライブテーマの横にあるカスタマイズボタンをクリック。
  3. テンプレートまたはセクションを選択：テーマエディタでページセレクタを使用してページテンプレートを選択（例: ホームページ、製品ページ）または左側のサイドバーからヘッダーを選択してバナーをサイトヘッダーに追加。
  4. セクションを追加：左側のサイドバーでセクションを追加（またはブロックを追加）をクリックし、AppsセクションからBundlezaバナーを検索して選択。
  5. セクションを設定：追加したら、セクション設定のドロップダウンから表示する特定のバナーを選択。
  6. 変更を保存：変更を適用するために保存をクリック。

ヒント

バナーはホームページでのプロモーション、製品ページでの特別オファー、コレクションページでのカテゴリ固有バナー、またはサイト全体の可視性のためのヘッダーに追加できます。
